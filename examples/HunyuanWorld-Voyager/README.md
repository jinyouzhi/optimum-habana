# inference
this is HunyuanWorld Voyager model example on Intel Gaudi.

1 install python package.
```bash
     pip install -r requirements.txt
```

2 download test data from:https://github.com/Tencent-Hunyuan/HunyuanWorld-Voyager/tree/main/examples to examples directory.

3 run demo script:
```bash
    bash run_hunyuan_voyager_demo.sh
```
